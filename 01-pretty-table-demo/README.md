# PrettyTable Demo

A simple demonstration of the PrettyTable library for creating formatted tables in Python.

## Description

This project showcases how to use the PrettyTable library to create nicely formatted tables with custom columns and alignment. The example creates a Pokemon table with names and types.

## Features

- Creates a formatted table with Pokemon data
- Demonstrates column alignment (center-aligned)
- Shows basic PrettyTable usage

## Requirements

```bash
pip install prettytable
```

## Usage

```bash
python main.py
```

## Output

The program will display a formatted table:

```
+-------------+----------+
| Pokemon Name|   Type   |
+-------------+----------+
|   Pikachu   | Electric |
|   Squirtle  |  Water   |
| Charamander |   Fire   |
+-------------+----------+
```

## Key Concepts

- **PrettyTable Library**: External library for table formatting
- **Column Management**: Adding columns with data
- **Alignment**: Setting table alignment properties
- **Data Presentation**: Organizing data in a readable format
